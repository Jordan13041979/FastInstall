💌 CONFIGURATION EMAIL
Suivez ces 5 étapes pour recevoir les devis sur Gmail.
ÉTAPE 1
Créer un compte EmailJS gratuit
Allez sur emailjs.com → cliquez "Sign Up" → créez un compte avec votre adresse Gmail. Gratuit jusqu’à 200 emails/mois.
ÉTAPE 2
Connecter votre Gmail
Dans EmailJS → "Email Services" → "Add New Service" → choisissez Gmail → connectez fast.install.meubles@gmail.com → cliquez "Create Service" → notez le Service ID (ex: service_abc123)
ÉTAPE 3
Créer le modèle d’email
Dans EmailJS → "Email Templates" → "Create New Template" → remplissez ainsi :
To (destinataire): fast.install.meubles@gmail.com
Subject: Nouveau devis - {{prenom}} {{nom}}
Body:
Prénom: {{prenom}}
Nom: {{nom}}
Téléphone: {{telephone}}
Email client: {{email}}
Service: {{service}}
Message: {{message}}
Date: {{date}}
Cliquez "Save" → notez le Template ID (ex: template_xyz789)
ÉTAPE 4
Copier votre clé publique
Dans EmailJS → "Account" → "General" → copiez la Public Key (ex: AbCdEfGhIjK12345)
ÉTAPE 5
Collez vos 3 clés ci-dessous
Public Key
Service ID
